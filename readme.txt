A search plugin that can be changed graphically writing HTML, both for the search bar and for the results

== Description ==

**Simple Html Search** let you decide whatever you need in your search bar, filtering by post title or custom fields and so forth,
you just need to write the HTML to customize it graphically.
Same thing for the results, you specify what you need writing HTML, and how to display it.
